# 源码解析

1、非常的简单粗暴，用3个正方形，每个旋转30°，形成夹角，即可形成齿轮

<div id="example1">
  <div></div>
  <div></div>
  <div></div>
</div>

#example1{
  position: relative;
  width: 150px;
  height: 150px;
  margin: 0 auto;
  div{
    position: absolute;
    width: 100px;
    height: 100px;
    background: #f98db9;
    top: 50%;
    left: 50%;
    &:first-child{
      transform: translate(-50%, -50%) rotate(0deg)
    }
    &:nth-child(2){
      transform: translate(-50%, -50%) rotate(30deg)
    }
    &:last-child{
      transform: translate(-50%, -50%) rotate(60deg)
    }
  }
}
